# graphql_with_react

Learn and master GraphQL by building real react apps with React and Node

## Some commands

`cd users`
`npm run json:server`
`npm run dev`

## URLS

### JSon server
`http://localhost:3000/`

### QraphiQL

`http://localhost:4000/graphql`
